FlexBox

Usado para posicionar elementos dentro de um layout

Pode ser escolhido trabalhar com linhas OU colunas

Dividir em duas categorias:

  • Container (elemento pai)
  • Items (elementos filhos)

  1. display: flex

    Indica que o container utilizará o posicionamento flexbox

  2. flex-direction

    Indica a direção que os items serão disponibilizados

  3. flex-wrap

    Indica se os items quebrarão de linha ao diminuir o container

  4. flex-flow

    Atalho para flex-direction + flex-wrap

  5. justify-content

    Usado para alinhar os items

  6. align-items

    Usado para alinhar os items

  7. align-content

    Usado para alinhar as linhas

  1. order:

    Escolhe a ordem do item dentro do container

  2. flex-grow:

    Determina o quanto o item irá crescer em comparação aos outros items do caontainer

  3. flex-shrink:

    Determina o quanto o item irá encolher em comparação aos outros items do container

  4. flex-basis:

    Tamanho inicial do item

  5. flex:

    Atalho para grow, shrink e basis

  6. align-self:

    Alinhamento do item na cruz. Sobrepõe a propriedade align-items do container.
