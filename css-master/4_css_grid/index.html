CSS Grid

O CSS grid oferece um sistema de layout em grade com linhas e colunas.

Facilitará o desenho de layouts sem utilizar floats e position.

Dividir em duas categorias:

  • Container (elemento pai)
  • Items (elementos filhos)

  1. grid-template-columns

    Indica quantas colunas terá o layout e o tamanho de cada coluna

  2. grid-template-rows

    Indica quantas linhas terá o layout e o tamanho de cada linha

  3. justify-content

    Alinha a grade inteira dentro do container

  4. align-content

  5. grid-template-areas

  1. grid-column (start, end)

    Indica em qual coluna o item começa e em qual coluna ele terminará

  2. grid-row (start, end)

    Indica em qual linha o item começa e em qual coluna o item terminará

  3. grid-area

    Atalho para grid-row e grid-column

  4. grid-area

    Nomeia o item para ser utilizado no grid-template-areas do container
