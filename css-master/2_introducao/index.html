Adicionando CSS na página

Existem algumas formas de utilizar a tecnologia CSS na página desejada. As principais, e mais nativas, são:

  • Inline: Atributo style de uma tag HTML (Má prática)
  • Tag style: Pode ser adicionada em qualquer ponto diretamente na página utilizada
  • Arquivo externo: Podemos importar o CSS por arquivo, sendo ele de dentro do projeto ou de algum host externo
Inline Tag Style Arquivo Externo

Observação: Nestes primeiros exemplos, iremos utilizar a tag style para melhor visualização. Quando ela é utilizada, normalmente terá apenas uma por página... aqui faremos várias para ficarem próximas aos elementos.

Seletor CSS

Utilizaremos o Seletor CSS para acessar os elementos do HTML e alterar seu estilo utilizando diversas propriedades nativas do CSS

As principais formas de acessar um elemento são:

  • Inline: Atributo style diretamente em uma tag HTML (Má prática)
  • Tag HTML: Faremos o CSS encontrar o elemento diretamente pelo nome da tag
  • Classe: Utilizando a propriedade class do elemento HTML. Exemplo: ".nome-da-classe"
  • Id: Utilizando a propriedade id do elemento HTML. Exemplo: "#id_do_elemento"
  • CSS Path: Podemos "andar" pelos seletores CSS de diversas formas utilizando o caminho dos elementos disponibilizados na página. Exemplo: ".minhaClasse > #meuId > div"
